如何使用

引入JS文件(CDN)

<script src="https://cdn.jsdelivr.net/gh/cqhcharlie/Auto-Theme-button/button_mode/theme-toggle.min.js"></script>
<script>
  new ThemeToggle();
</script>

全部配置选项

参数名 类型 默认值 说明
position String 'top-right' 按钮位置。可选值：'top-left'、'top-right'、'bottom-left'、'bottom-right'、'fixed'（页面中央）
offset Object { x: 20, y: 20 } 按钮距离边缘的偏移量（像素）。x 为水平偏移，y 为垂直偏移
size Number 50 按钮大小（像素）。建议范围：30-100
darkColor String '#1a1a1a' 深色模式的背景颜色。支持任何有效的 CSS 颜色值
lightColor String '#ffffff' 浅色模式的背景颜色。支持任何有效的 CSS 颜色值
transition String '0.3s ease' 主题切换的过渡动画效果。CSS transition 属性值
zIndex Number 9999 按钮的 CSS z-index 层级，确保按钮显示在其他元素上方
defaultTheme String 'system' 默认主题。可选值：'light'（浅色）、'dark'（深色）、'system'（跟随系统）。仅在用户首次访问时生效
excludeSelectors Array [] 排除深色化的元素选择器数组。这些元素将始终保持浅色样式。例如：['.no-dark', '#logo', '.card.important']
useFilter Boolean false 是否启用“过滤器深色模式”（实验性）。当某些网站难以整体深色化时，开启此项可通过 invert + hue-rotate 实现快速深色化。开启后 excludeSelectors 将被忽略，改用 filterExcludeSelectors。
filterValue String 'invert(1) hue-rotate(180deg)' 过滤器深色模式所使用的 CSS filter 值，可根据需要进行微调。
filterExcludeSelectors Array ['img','video','picture','canvas','iframe','svg'] 当 useFilter=true 时，这些选择器会被“反向再反向”，以保持原始颜色（例如图片、视频等媒体）。

完整配置示例

new ThemeToggle({
  position: 'top-right',
  offset: { x: 20, y: 20 },
  size: 50,
  darkColor: '#1a1a1a',
  lightColor: '#ffffff',
  transition: '0.3s ease',
  zIndex: 9999,
  defaultTheme: 'dark',  // 'light', 'dark' 或 'system'
  excludeSelectors: ['.no-dark', '#keep-light'],
  // 过滤器深色模式（当某些站点难以整体变暗时使用）
  useFilter: false, // true 开启
  filterValue: 'invert(1) hue-rotate(180deg)',
  filterExcludeSelectors: ['img', 'video', 'picture', 'canvas', 'iframe', 'svg']
});

EventListener监听主题变化事件

document.addEventListener('themeChanged', (e) => {
  console.log('主题已切换为:', e.detail.theme);
  // 在这里添加你的自定义逻辑
});
